# STM32 Nano Development Board

A compact and high-performance **STM32 Nano Development Board** designed in **Altium Designer** for embedded prototyping, learning, and rapid firmware development. This board uses a **4-layer PCB stack-up** for excellent signal integrity and power stability.

---

## üîß Key Features

* STM32 Microcontroller (Nano form factor)
* **4-Layer PCB Design** for high reliability
* Onboard **3.3V LDO Voltage Regulator**
* **External Crystal Oscillator** for accurate system clock
* **SWD Programming & Debug Interface**
* **Reset Push Button** with proper protection
* **Status LEDs** for power and activity indication
* **Full GPIO Breakout Headers**
* Compact, breadboard-friendly layout

---

## üß± PCB Stack-Up (4 Layer)

| Layer         | Purpose                                |
| ------------- | -------------------------------------- |
| Top Layer     | High-speed signal routing & components |
| Inner Layer 1 | **Solid Ground Plane (GND)**           |
| Inner Layer 2 | **Dedicated Power Plane (3.3V)**       |
| Bottom Layer  | Signal routing                         |

‚úÖ This stack-up ensures:

* Low noise and EMI reduction
* Excellent return paths for high-speed signals
* Stable and clean power distribution
* Better thermal and electrical performance
